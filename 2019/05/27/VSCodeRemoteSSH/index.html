VS Codey遠端開發套件：Remote-SSH

2019-05-27
Mars Hung
dev
文章閱讀量次

說明

使用VS Codey套件Remote-SSH實現遠端開發

安裝

  1. 下載並安裝VS Code

  2. 安裝VS Code套件Remote - SSH

  3. 建立SSH Key

  4. 設定SSH config
    In VS Code, run Remote-SSH: Open Configuration File

  5. 安裝SSH Key至遠端

  6. 連線至遠端
    In VS Code, run Remote-SSH: Connect to Host

    第一次連線會自動安裝遠端套件

  7. 本地端環境設定

    按[ctrl+,]選User

  8. 遠端環境設定

    按[ctrl+,]選Remote
